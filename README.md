## Features

+ Authorization (Register, Login, Logout)
+ Autologin / Autologout
+ Validate fields
+ Create tests with parameters (name and quantity answers)

## Used scripts

+ React (with react-redux and react-router-dom)
+ Redux (with redux-thunk)
+ Axios
+ Firebase
